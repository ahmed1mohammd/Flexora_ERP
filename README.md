# 🚀 Flexora - Landing Page Portal

صفحة الهبوط والتسجيل الفاخرة والمؤتمتة بالكامل (Luxury Gym Automation SaaS Landing Page) لمنصة **فليكسورا** في جمهورية مصر العربية.

---

## 🎨 المميزات التقنية والبصرية
- **تصميم متجاوب بالكامل (Mobile-First):** متوافق تماماً مع جميع الهواتف الذكية والأجهزة اللوحية دون أي شريط تمرير أفقي.
- **هوية بصرية نيونية فاخرة:** تجمع بين اللون الأسود السينمائي وتوهج اللون الأحمر الياقوتي (Crimson) بتأثيرات حركة دقيقة ومريحة للعين.
- **محرك هيدرة الأسعار الديناميكي:** يقوم بجلب أسعار خطط الاشتراك ومميزاتها ديناميكياً من السيرفر، مع وجود نظام طوارئ احتياطي تلقائي (Robust Fallback) في حال عدم توفر اتصال بالشبكة.
- **بوابة الحجز المباشر النخبوية:** تتيح لمالكي الصالات الرياضية حجز نسختهم التجريبية وتأكيد طلب التفعيل المبدئي خلال 24 ساعة.

---

## 🛠️ التقنيات المستخدمة
- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript
- SweetAlert2 (للتنبيهات الاحترافية المتناسقة مع هوية الموقع)
